# Tapping of the Undead

Brush up on your morse code skills by exterminating horizontally-spinning rats in this game.

## Controls
|Key|Action|
|---|---|
|<kbd>WASD</kbd>| Switch target |
|<kbd>i</kbd>| Dit (short burst) |
|<kbd>j</kbd>| Dah (long tone) |
